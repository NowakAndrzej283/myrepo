# Package Managment on Debian/Ubuntu based Distributions

sudo apt update - this is going to update your repository

apt search - we are searching the indexes for packeges 

# When we want to install something we have to use "sudo" because it is going to make changes in th overall system

sudo apt install nameofthepackage  - we are installing a wanted package 

# Removing a package

sudo apt remove nameofthepackage  - we are removing the package

sudo apt autoremove - we are removing the packages that were installed with the main package 

# Upgrading packages

sudo apt upgrade - upgrading all packages

sudo apt dist-upgrade -we use that command after the previous one to upgrade everything that wasn't installed yet

sudo apt reboot - reboots the machine
